图片轮播组件图片轮播，采用跑马灯效果
使用说明
名字 类型 默认值 说明
配置参数
pictures Array [] 轮播图片资源及自定义内容
pictureWidth Number 500 图片显示宽度
pictureHeight Number 200 图片显示高度
content String "" 图片内容模板
effect String "slide" 图片切换类型，取值：none:无特效 / fade:渐隐 / slide:滑动
easing String "easeInOut" 缓动类型，取值 linear:无缓动效果 / easeIn:在过渡的开始提供缓动效果 / easeOut:在过渡的结尾提供缓动效果 / easeInOut 在过渡的开始和结尾提供缓动效果
timeout Number 2500 切换时间间隔
during Number 300 切换速度，越小越快，单位为毫秒
alwaysShowArrow Boolean true 显示左右切换箭头
alwaysShowSelection Boolean true 显示底部圆形切换部件
autoSlide Boolean true 自动播放
hoverStop Boolean true 鼠标经过停止播放
adaptiveWidth Boolean false 适应外围宽度，为true时指定pictureWidth不起作用
adaptiveHeight Boolean false 适应外围高度，为true时指定pictureHeight不起作用
eventType String "click" 触发tab切换的nav上的事件类型，取值click\mouseenter\both
arrowLeftNormalSrc String "" 左箭头正常状态图标，可不传
arrowRightNormalSrc String "" 右箭头正常状态图标，可不传
arrowLeftHoverSrc String "" 左箭头hover状态图标，可不传
arrowRightHoverSrc String "" 右箭头hover状态图标，可不传
arrowLeftClass String "" 左右箭头的className，可不传
arrowRightClass String "" 左右箭头的className，可不传

  • 图片轮播组件-自定义宽高
  • 图片轮播组件-自定义图片切换时间间隔 / 自定义图片切换速度
  • 图片轮播组件-自定义不显示左右切换箭头和底部圆形选择部件 / 自定义鼠标经过不停止播放
  • 图片轮播组件-自定义effect
  • 图片轮播组件-自定义缓动类型
  • 图片轮播组件-自定义填充外围宽度和高度
  • 图片轮播组件-自定义图片内容模板
